# Boas Práticas de Segurança

- Use segredos fortes e faça rotação regularmente
- Restrinja as origens de CORS em produção
- Valide os uploads antes de persistir
- Sirva a aplicação atrás de HTTPS
- Mantenha os tokens com vida curta
- Revise o acesso ao runner do GitHub e às credenciais do registry

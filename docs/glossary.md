# Glossário

| Termo | Significado |
|---|---|
| Backstage | Plataforma de portal do desenvolvedor e catálogo de software |
| Scaffolder | Componente do Backstage que cria software a partir de templates |
| Template | Entidade do Backstage que descreve como gerar um repositório |
| Component | Entidade do Backstage que descreve um componente de software em execução |
| System | Entidade do Backstage usada para agrupar componentes relacionados |
| JWT | JSON Web Token usado para autenticação bearer |
| Alembic | Ferramenta de migração de banco para projetos SQLAlchemy |
| Helm | Gerenciador de pacotes para Kubernetes |
| ArgoCD | Controlador GitOps para Kubernetes |

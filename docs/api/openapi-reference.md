# Referência OpenAPI

## Documentação nativa

O FastAPI gera automaticamente a especificação OpenAPI da aplicação.

## URLs úteis

- Swagger UI: `/docs`
- ReDoc: `/redoc`
- OpenAPI JSON: `/openapi.json`

## Por que isso importa

- Permite explorar a API em tempo real
- Pode ser importado por ferramentas de geração de SDK de cliente
- É a forma mais rápida de inspecionar schemas de request e response

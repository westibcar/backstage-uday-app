# Exemplos de Uso

## Criar um serviço a partir do Backstage

1. Abra o template FastAPI no Backstage.
2. Preencha o nome da aplicação, owner, URL do repositório e namespace.
3. Adicione os valores de infraestrutura.
4. Envie o formulário.
5. O Scaffolder publica o repositório e registra o componente.

## Executar localmente

```bash
docker compose up --build -d
```

## Verificar a API

```bash
curl http://localhost:8000/health
```

## Registrar e fazer login

Use os formulários do frontend ou os exemplos de API da próxima página.
